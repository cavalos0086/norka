SKETCHES

Before making any digital design, I decided to make rough sketches of the main pages to get the client's approval before moving on to digitalized sketches.'

Landing Page:

User gets a welcome to this website.

Home Page:

Users are able to add a task or view past tasks.

Adding a task page:

Users are able to enter a title and how many pomodoros.

View my tasks page:

Users see the lists of their tasks.

Timer running page:

Users are able to see the title and how many pomodoros while the timer is running.

LOW-FIDELITY WIREFRAMES

Once these sketches were presented, I made a few changes based on the client's suggestion and digitalized them on Sketch

1

The "Next" button was removed from the landing page

2

A Side menu was added with the two main features

3

A "Description" field was added when the user creates a new task

4

Tool icon was added next to the tasks. Users are able to delete, edit, or start the timer

5

A pop-up window appears when the timer is started showing the pomodor task details
