Tools Used

Target Device

The pomodoro technique is a time management method that uses a timer to break down work into intervals, and usually its separated by short breaks. These short breaks are called POMODOROS.
Design and test a web app that enables users to add, edit, view a list of To-Do items, start a pomodoro timer and get alerts when the pomodoro is over.
